Content Packaging Resources
IMS Content Packaging is a specification for sending learning resources (or learning objects) from one program to another, facilitating easier delivery, reuse and sharing of materials.

Content Packaging Briefing
CETIS have produced a briefing paper on Content Packaging which provides a concise introduction to the specification. Download the CETIS CP Briefing here: http://www.cetis.ac.uk/lib/media/CPbreif.pdf

Content Packaging Interactive Documentation
CETIS have provided a site offering Interactive Documentation for the Content Packaging specification. You can see the Interactive Documentation site here: http://ford.ces.strath.ac.uk/IMSCP/
