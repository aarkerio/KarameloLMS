Metadata Resources
The IMS Metadata specification was one of the first to be produced and as such it is well supported. The best place to start is probably the Metadata SIG section of the CETIS web site: http://metadata.cetis.ac.uk/

Metadata Guide
One of the most useful resources on the Metadata SIG site is the http://metadata.cetis.ac.uk/guides/guide.doc.

UK Learning Object Metadata Framework (draft)
Previously referred to as the UK Common Metadata Framework, this is a set of guidelines to inform UK practitioners wishing to implement the IEEE LOM in such a way as to provide maximum benefit for the UK educational community.http://www.cetis.ac.uk/profiles/uklomcore.
