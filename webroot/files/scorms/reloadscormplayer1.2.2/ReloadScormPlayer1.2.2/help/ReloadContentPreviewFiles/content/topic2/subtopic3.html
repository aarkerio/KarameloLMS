Uninstalling (Microsoft Windows)
To remove RELOAD SCORM Player from your system (and prior to upgrading to a new version of the software), click the 'Uninstall Reload SCORM Player' Icon in the RELOAD program group.

If you chose not to create a Program Group, you will find the relevant file at: C:\Program Files\ReloadTools\ScormPlayer\UninstallerData\Uninstall Reload SCORM Player.exe or wherever you installed to.

On uninstall, files and folders you have created in the RELOAD SCORM Player Installation directory will not be removed.

Uninstalling (Macintosh OSX)
To remove RELOAD SCORM Player from your system (and prior to installing a new copy), locate the installed software and remove the folder - this should contain the RELOAD SCORM Player executable, and a folder containing Open Source Licenses). All files can be safely removed by dragging to the Trash, but check that you didn't create any files of your own in the Install directory.
