Conventions
In these Help files, a number of conventions will be followed:

  • All screenshots will be taken from Windows XP: the look of the program may vary slightly on different platforms, but all menus and icons should be in the same place, look similar and function in the same way.
  • Anything you need to type in will be in a monospaced font.
  • Any actions you need to perform will be described in bold.
